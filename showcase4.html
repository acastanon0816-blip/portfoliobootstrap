Project Overview

This projects showcased on this page were created in Adobe Animate and include a banner animation, a character design project, and a product advertisement. Together, these projects demonstrate my understanding of the fundamentals of the Animate workspace.

Animate Project 1: Banner Project

The purpose is to design an interactive banner for the homepage of the company’s online retail store that will help customers learn more about the shop’s history, commitment to giving expert advice, its values/mission (About Us), and its product lines.

Skills and Tools Used

  • Timeline organization with layers and labeled frames
  • Keyframes to control changes over time
  • Motion tweening for smooth movement
  • Exporting a final animation for web display

Animate Project 2: Character Design

The purpose is to design an original character utilizing basic illustration principles, conversion techniques, and motion presets to give the character a simple movement to accomplish.

Skills and Tools Used

  • Drawing and shape tools for building character parts
  • Color fills and strokes for consistent style
  • Symbols to reuse assets and keep the file organized
  • Layer management for clean editing

Animate Project 3: Product Advertisement

The purpose is to design a product ad for social media promotions utilizing a specific brand carried in the shop. The design will implement basic animation principles, image imports, conversion techniques, and motion presets to add simple animations to draw the shopper’s attention..

Skills and Tools Used

  • Keyframes and tweens to animate text and objects
  • Basic easing/timing adjustments for smoother motion
  • Scene structure and timeline planning
  • Export settings for sharing the finished ad

Challenges and Improvements

A challenge across these projects was timing and pacing—making sure movement feels smooth and the viewer has enough time to read the message. These projects helped me practice planning animations on the timeline and keeping layers organized. Next time, I would experiment with more advanced easing and refine transitions for an even more polished result.
